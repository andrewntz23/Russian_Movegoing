CINEMA and MEMORY in St. Petersburg

About

This project is for the Fall 2015 course "Digital Humanities."

Project Objectives

Using computational tools, the Project examines oral history interviews, in which interviews reconstruct their moviegoing experiences in St. Petersburg, Russia.

Specifically, the Project analyzes

  • what moviegoers remember (films, actors, directors),
  • how they evaluate films, actors, characters,
  • how they define themsleves as narrators (I or we), and
  • what they hesitate about when they recollect their moviegoing experiences.

Project Scope

For this pilot project we use nine oral history interviews.

The Future of the Project

We plan to expand the corpus of interviews. Also we will use more statistical and social networking tools.
